# job4j_sql

Проект Job4j([Job4j](https://job4j.ru/))_sql

Предназначен для тестирования и хранения заданий по курсу Job4j_SQL(only).

курс состоит из:
- основы SQL;
- настройка программного окружения и знакомства с программами;
- Работа в psql и DBeaver/IDEA Database
-  DDL: таблицы, типы данных и ограничения
-  DML: insert, update, delete

* Разделы и блоки будут открываться и дополлнятся по мере открытия задач и проекта.*
